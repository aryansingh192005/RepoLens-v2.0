# RepoLens Design System v1

## Product Vision

RepoLens is an AI-powered developer platform that helps developers understand GitHub profiles, repositories, and open source projects faster.

The interface should feel:

- Fast
- Clean
- Professional
- Minimal
- Modern
- Calm
- Trustworthy

Not flashy.

# UX Rules

Every action should have feedback.

Never show a blank page.

Every loading state must have a skeleton.

Every error must explain what happened.

Primary CTA should always be visible.
